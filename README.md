# vue-table

## Инициализация проекта
```
npm install
```

### Запуск в режиме разработки
```
npm run serve
```

### Сборка
```
npm run build

Сборку запустить, просто открыв файл index.html, не получится.
Нужен локальный HTTP сервер. Например, serv:

npm install -g serve

Запуск:

serve -s dist
```
